# Premium Legal UI System

Design this project with a premium legal-tech aesthetic.

## Brand Identity

This interface must feel:

- premium
- trustworthy
- elegant
- modern
- minimal
- executive
- luxury without excess

Avoid:

- startup-style gradients
- neon colors
- generic SaaS design
- childish rounded UI
- excessive animations
- overly playful interfaces

## Color System

Primary: #A83232
Accent: #D9C7A3
Secondary Text: #2F2F2F
Background: #FAF8F5
Borders: #E8E2D8
Success: #4F6B4A

## Typography

Avoid Inter.

Prefer:

- DM Serif Display
- Plus Jakarta Sans
- Instrument Sans
- General Sans
- IBM Plex Sans

## Layout Rules

Use:

- strong whitespace
- editorial composition
- refined hover states
- subtle shadows
- premium cards
- elegant spacing

Avoid:

- generic SaaS cards
- purple gradients
- excessive border radius
- startup hero sections
- childish UI patterns
